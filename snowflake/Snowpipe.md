# Automating Snowpipe Using Cloud Messaging
- 클라우드 저장소에 새로운 파일일 도착했다는 메시지를 snowpipe로 전달

# What is Snowpipe?
- Snowpipe는 데이터 공급도구
- 외부의 어플리케이션이 준 실시간으로 데이터를 생성해서 외부 저장소에 데이터 저장
- 외부 저장소에 저장된 데이터를 Snowpipe를 통해서 Snowflake 테이블로 저장
- 내부, 외부 데이터 저장소에 모두 사용 가능
