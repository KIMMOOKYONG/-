# 지도 시각화 도구 Folium 사용법
- 매뉴얼: https://python-visualization.github.io/folium/

# folium 개요
- folium은 leaflet.js 기반으로 만들어진 Python 지도 시각화 라이브러리

# folium 설치
```
!pip install folium
```

# 기본 좌표 설정
- location: 위, 경도 좌표
- zoom_start: 지도 레벨, 확대 정도(최대 18)

# 마커 추가
- location: 위도/경도 좌표
- popup: 팝업 문구(클릭)
- tooltip: 툴팁 문구(마우스 오버)
