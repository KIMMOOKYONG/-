# Rule-Based(규칙 기반) 챗봇 만들기
# Rule-Based 설계하기
- Intent(의도)
- Entity(개체)
- 대화의 규칙을 직접 설계해서 어떤 말이 만들어진 규칙에 해당한다면 답변하는 방식이다.
